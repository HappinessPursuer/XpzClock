# 我的一个Android组合控件：XpzClock

# 在Android项目中快速使用：
（1）配置settings.gradle 
在Repositories中添加：
  maven { url 'https://jitpack.io' }
如：
```
repositories {
        google()
        mavenCentral()
        maven { url 'https://jitpack.io' 
        }
```
（2）配置build.gradle
在build.gradle中添加：
  implementation 'com.github.HappinessPursuer:XpzClock:v1.1'
如：
```
dependencies {
    implementation 'com.github.HappinessPursuer:XpzClock:v1.1'
    }
```
